<p class="mb-4">「頑張らない人のためのタスク管理」をコンセプトにした、ブラウザ上で動作するシンプルなタスク管理アプリケーションです。</p>

<h4 class="font-bold text-appleDark mb-3 mt-8">アプリケーションの特長</h4>
<ul class="space-y-4 text-sm text-appleDark/80 bg-black/5 p-5 rounded-xl border border-black/10">
  <li class="flex items-start">
    <i class="fa-solid fa-face-smile text-appleBlue mt-0.5 mr-3 w-4 text-center shrink-0"></i>
    <div><strong class="text-appleDark block mb-1">やさしい設計</strong>未完了のタスクは翌日に自動で持ち越されます。タスクをこなせなかった日があっても問題ありません。</div>
  </li>
  <li class="flex items-start">
    <i class="fa-solid fa-clock-rotate-left text-appleBlue mt-0.5 mr-3 w-4 text-center shrink-0"></i>
    <div><strong class="text-appleDark block mb-1">過去にとらわれない</strong>1週間以上経過した完了済みタスクは自動で削除されます。これにより、常に「今」のタスクに集中できる設計になっています。</div>
  </li>
  <li class="flex items-start">
    <i class="fa-solid fa-stopwatch text-appleBlue mt-0.5 mr-3 w-4 text-center shrink-0"></i>
    <div><strong class="text-appleDark block mb-1">ざっくりとした時間管理</strong>分単位の細かいスケジュール管理ではなく、1日を「3時間ブロック」に分割し、そこにタスクを放り込むだけで直感的に管理できます。</div>
  </li>
  <li class="flex items-start">
    <i class="fa-solid fa-mobile-screen text-appleBlue mt-0.5 mr-3 w-4 text-center shrink-0"></i>
    <div><strong class="text-appleDark block mb-1">アプリとしてインストール可能（PWA対応）</strong>スマートフォンやPCのブラウザから、ネイティブアプリのように端末へインストールすることが可能です。</div>
  </li>
</ul>
