<p class="mb-4">学校では毎年、未使用のままや、ほとんど使われなかった教科書が大量に廃棄されていた。一方で、教材が高くて買えないという声が実際に身近にあることも目の当たりにしていた。</p>

<p class="mb-4">この「もったいない」と「必要としている人がいる」という二つの現実をつなぎ合わせられないかと考え、中高の仲間3人でSTARCLEをはじめた。</p>

<h4 class="font-bold text-appleDark mb-3 mt-8">仕組み</h4>

<p class="mb-4">教材の寄付を呼びかけ、寮の空き部屋を倉庫として借りて在庫を管理した。当初は本体価格を無料とし、レターパックの送料のみで全国に送付していた。その後、ボランティアコンテストで得た賞金を活動資金に充てることで、送料も含めて完全無料での配送を実現した。実際に全国から注文を受け、数十〜百冊台の教材を届けた。</p>

<img src="/portfolio/img/starcle/inventory-prep.png" alt="在庫の準備の様子" class="w-full rounded-xl shadow-md my-6 object-cover" />

<h4 class="font-bold text-appleDark mb-3 mt-10">自分の担当</h4>

<p class="mb-4">3人での運営のなかで、自分は主にWeb周りを担当した。</p>

<ul class="space-y-5 text-sm text-appleDark/80 bg-black/5 p-5 rounded-xl border border-black/10 mt-4">
  <li class="flex items-start">
    <i class="fa-solid fa-globe text-appleBlue mt-0.5 mr-3 w-4 text-center shrink-0"></i>
    <div>
      <strong class="text-appleDark block mb-1">Webサイト・注文システムの構築</strong>
      当時はまだ生成AIが存在しなかったため、HTML/CSSを素で直打ちしてサイトを制作した。個人情報を扱う注文周りは、セキュリティリスクを考慮して外部ツールを活用した。
    </div>
  </li>
  <li class="flex items-start">
    <i class="fa-solid fa-magnifying-glass text-appleBlue mt-0.5 mr-3 w-4 text-center shrink-0"></i>
    <div>
      <strong class="text-appleDark block mb-1">命名・SEO・ロゴ制作</strong>
      「STARCLE」という名称は自分が命名した。「STudy Aid ReCycLE」の頭文字から作り、Google検索上の競合の少なさ、ドメインの空き状況、カタカナ読みしたときの語呂の良さを加味して決定した。ロゴマークも自分が作成した。
    </div>
  </li>
  <li class="flex items-start">
    <i class="fa-solid fa-server text-appleBlue mt-0.5 mr-3 w-4 text-center shrink-0"></i>
    <div>
      <strong class="text-appleDark block mb-1">ドメイン・Webサーバの管理</strong>
      独自ドメイン（starcle.jp / starcle.org）の取得から、レンタルサーバのセットアップ・維持管理を担った。現在はドメイン・サーバの失効によりサイトは現存しないが、当時のアーカイブが残っている。
    </div>
  </li>
  <li class="flex items-start">
    <i class="fa-solid fa-video text-appleBlue mt-0.5 mr-3 w-4 text-center shrink-0"></i>
    <div>
      <strong class="text-appleDark block mb-1">コンペ応募用の動画制作</strong>
      受賞に向けたボランティアコンテストへの応募動画の制作も担当した。
    </div>
  </li>
</ul>
