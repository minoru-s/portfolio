<p class="mb-4">高校2年生の時期に、二つのビジネスコンテストへ出場した記録。いずれも入賞には至らなかったが、自分の考えをビジネス提案としてまとめ、審査・発表を経験した機会として残しておく。</p>

<h4 class="font-bold text-appleDark mb-3 mt-8">キャリア甲子園</h4>

<p class="mb-4">全国の高校生を対象としたビジネスアイデアコンテスト。</p>

<ul class="space-y-4 text-sm text-appleDark/80 bg-black/5 p-5 rounded-xl border border-black/10">
  <li class="flex items-start">
    <i class="fa-solid fa-lightbulb text-appleBlue mt-0.5 mr-3 w-4 text-center shrink-0"></i>
    <div>
      <strong class="text-appleDark block mb-1">提案テーマ：ダイナミックプライシングの一般化</strong>
      航空券やホテルなど一部の業界では普及しているダイナミックプライシング（需給に応じて価格が変動する仕組み）を、より多くの業種・場面で活用できるようにする仕組みについて提案した。
    </div>
  </li>
</ul>

<h4 class="font-bold text-appleDark mb-3 mt-10">TSG（Tokyo Startup Gateway）</h4>

<p class="mb-4">東京都が主催する高校生・大学生向けの起業家育成プログラム。</p>

<ul class="space-y-4 text-sm text-appleDark/80 bg-black/5 p-5 rounded-xl border border-black/10">
  <li class="flex items-start">
    <i class="fa-solid fa-lightbulb text-appleBlue mt-0.5 mr-3 w-4 text-center shrink-0"></i>
    <div>
      <strong class="text-appleDark block mb-1">提案テーマ：情報の信頼性と民意を可視化するサービス</strong>
      氾濫する情報の中で何が正しいかを判断しにくい現状に対し、民意を定量的に図ることでその信頼性を可視化するサービスを提案した。第一関門を突破し、最終ステージの手前まで勝ち進んだ。
    </div>
  </li>
</ul>
